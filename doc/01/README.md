## プレミアム会員というのが出来た
+ この会員は全コースを1000円引きで利用出来る

```Bash
$ php controller.php 

your account? normal/premium : normal
plan? small/normal/large : small
選択したのはスモールプランで月3000円、容量は1GBです。


$ php controller.php

your account? normal/premium : premium
plan? small/normal/large : small
選択したのはスモールプランで月2000円、容量は1GBです。
```
